# JustCall

JustCall integration enables agents to access phone call logs, SMS message history, and communication analytics from JustCall. The integration uses JustCall's MCP server, providing standardized tool access to communication data for CRM integration and compliance tracking.

## Overview

The JustCall integration exposes communication operations as MCP tools that agents can call during conversations. Agents can search call logs, retrieve SMS messages, access call recordings, and query communication history—all while respecting user permissions and compliance requirements.

<Callout type="info">
    JustCall uses a self-hosted MCP server that communicates with JustCall's API. The integration
    requires an auth token (API key:secret format) and handles authentication automatically.
</Callout>

## Prerequisites

- JustCall account with API access
- API credentials (API key and secret)
- Understanding of communication compliance (TCPA, GDPR)

## Setup

### Step 1: Get JustCall API credentials

1. Log in to [JustCall Dashboard](https://app.justcall.io)
2. Navigate to **Settings** > **API**
3. Generate API credentials:
    - **API Key**: Your API key
    - **API Secret**: Your API secret
4. Format: `api_key:api_secret` (colon-separated)

<Callout type="warning">
    JustCall auth token uses `api_key:api_secret` format. Store securely—never commit to version
    control.
</Callout>

### Step 2: Configure in AgentC2

**Environment variables:**

```bash
JUSTCALL_AUTH_TOKEN="api_key:api_secret"
```

**Via UI:**

1. Navigate to **Settings > Integrations**
2. Find **JustCall** and click **Connect**
3. Enter your auth token (format: `api_key:api_secret`)
4. Click **Test Connection** to verify credentials
5. Tools are automatically discovered and available

**Via API:**

```bash
curl -X POST https://agentc2.ai/agent/api/integrations/connections \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "providerKey": "justcall",
    "name": "Production JustCall",
    "credentials": {
      "JUSTCALL_AUTH_TOKEN": "api_key:api_secret"
    }
  }'
```

### Step 3: Attach tools to agents

List available JustCall tools:

```bash
curl https://agentc2.ai/agent/api/tools \
  -H "Authorization: Bearer $TOKEN" \
  | jq '.tools[] | select(.id | startswith("justcall-"))'
```

Attach tools to an agent:

```bash
curl -X POST https://agentc2.ai/agent/api/agents/sales-assistant/tools \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "toolId": "justcall_justcall-get-call-logs"
  }'
```

## Available tools

| Tool Name                                 | Description                                             |
| ----------------------------------------- | ------------------------------------------------------- |
| `justcall_justcall-get-call-logs`         | Get call logs with metadata (date, duration, direction) |
| `justcall_justcall-get-call-details`      | Get detailed call information including recording URL   |
| `justcall_justcall-get-sms-messages`      | Get SMS message history                                 |
| `justcall_justcall-send-sms`              | Send SMS message (if supported)                         |
| `justcall_justcall-search-communications` | Search calls and SMS by phone number, date, etc.        |

Tool names follow the pattern `justcall_{tool-name}`. Use the tools API to see the full list with parameter schemas.

## Common patterns

### Get call logs

```typescript
// Agent receives: "Show me calls from today"
// Agent calls:
{
  tool: "justcall_justcall-get-call-logs",
  arguments: {
    startDate: "2025-02-18",
    endDate: "2025-02-18",
    limit: 20
  }
}
// Returns: Array of calls with phone numbers, durations, timestamps
```

### Get call details

```typescript
// Agent receives: "Get details for the call with john@example.com"
// Agent calls:
{
  tool: "justcall_justcall-search-communications",
  arguments: {
    phoneNumber: "+1234567890",
    limit: 1
  }
}
// Then:
{
  tool: "justcall_justcall-get-call-details",
  arguments: {
    callId: "call-id-from-search"
  }
}
// Returns: Call details including recording URL, transcript (if available)
```

### Search SMS messages

```typescript
// Agent receives: "Find SMS messages with John"
// Agent calls:
{
  tool: "justcall_justcall-get-sms-messages",
  arguments: {
    phoneNumber: "+1234567890",
    startDate: "2025-02-01",
    endDate: "2025-02-18",
    limit: 10
  }
}
// Returns: Array of SMS messages with content, timestamps, direction
```

### Search communications

```typescript
// Agent receives: "Find all communications with +1234567890"
// Agent calls:
{
  tool: "justcall_justcall-search-communications",
  arguments: {
    phoneNumber: "+1234567890",
    startDate: "2025-02-01",
    endDate: "2025-02-18"
  }
}
// Returns: Combined calls and SMS for the phone number
```

## Communication compliance

### TCPA compliance

JustCall helps with TCPA (Telephone Consumer Protection Act) compliance:

- Consent tracking
- Do-not-call list management
- Opt-out handling

<Callout type="warning">
    Always ensure compliance with TCPA, GDPR, and local regulations when accessing or using
    communication data. Verify consent before accessing call logs or SMS.
</Callout>

### Data retention

JustCall retains communication data per plan:

- Check retention policies in JustCall settings
- Export data before retention expires if needed
- Respect user privacy and data protection regulations

### Access controls

JustCall respects user permissions:

- Users can only access communications they're authorized to view
- API keys are scoped to the account
- Team members may have restricted access

## Troubleshooting

| Problem                           | Solution                                                                                                    |
| --------------------------------- | ----------------------------------------------------------------------------------------------------------- |
| **"Authentication failed"**       | Verify `JUSTCALL_AUTH_TOKEN` format is `api_key:api_secret`. Check credentials are correct and not expired. |
| **"Call not found"**              | Verify call ID is correct. Check user has access to the call (authorized team member).                      |
| **"Permission denied"**           | User lacks permission to access communication data. Verify API key has required scopes.                     |
| **"Recording not available"**     | Some calls may not have recordings (not enabled, failed). Check call status and recording settings.         |
| **"Rate limit exceeded"**         | JustCall limits API calls per plan. Upgrade plan or implement rate limiting/queuing.                        |
| **"Invalid phone number format"** | Use E.164 format: `+1234567890`. Include country code and no spaces or dashes.                              |

## Rate limiting

JustCall API rate limits vary by plan:

- **Starter**: Limited requests/month
- **Professional/Enterprise**: Higher limits

Monitor usage:

```bash
curl https://api.justcall.io/v1/usage \
  -H "Authorization: Basic $(echo -n 'api_key:api_secret' | base64)"
```

## Best practices

### 1. Respect privacy

Always respect communication privacy:

```typescript
// Only access communications user is authorized to view
// Don't expose sensitive information in agent responses
const call = await callTool("justcall_justcall-get-call-details", { callId });
// Redact sensitive information if needed
```

### 2. Handle missing recordings gracefully

Not all calls have recordings:

```typescript
try {
    const call = await callTool("justcall_justcall-get-call-details", { callId });
    if (call.recordingUrl) {
        return `Call recording available: ${call.recordingUrl}`;
    } else {
        return "Call recording not available for this call.";
    }
} catch (error) {
    return "Could not retrieve call details.";
}
```

### 3. Use date filters

Limit results with date filters:

```typescript
{
  tool: "justcall_justcall-get-call-logs",
  arguments: {
    startDate: "2025-02-01",
    endDate: "2025-02-28",
    limit: 50
  }
}
```

### 4. Cache frequently accessed data

Cache call logs to reduce API calls:

```typescript
const cacheKey = `justcall:calls:${phoneNumber}:${date}`;
const cached = await cache.get(cacheKey);
if (cached) return cached;

const calls = await callTool("justcall_justcall-get-call-logs", { phoneNumber, date });
await cache.set(cacheKey, calls, 3600); // Cache 1 hour
```
